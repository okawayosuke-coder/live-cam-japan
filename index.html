YouTube Live

Google Cloud Console で「YouTube Data API v3」を有効化→APIキーを作成（無料枠あり）。ブラウザ用にHTTPリファラ制限の設定を推奨。

Windy Webcams

api.windy.com/keys で無料キーを取得。ブラウザ直fetchはCORSで弾かれる場合があり、その時は tools/fetch-windy.mjs で生成した data/windy.json を使います。
